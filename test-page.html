TodoList Chrome Extension 測試頁面

🧪 擴展功能測試

這個頁面用來測試 TodoList Chrome Extension 的文字選取功能。

測試步驟：
  1. 確保已在Chrome中載入擴展程序（使用 dist 資料夾）
  2. 在下方的藍色區塊中選取任意文字
  3. 觀察是否出現通知
  4. 點擊擴展圖標檢查待辦清單

📝 可選取的測試文字

測試文字 1：買牛奶和麵包
測試文字 2：準備明天的會議資料
測試文字 3：這是一段較長的測試文字，用來測試擴展是否能正確處理較長的文字內容並正確顯示在待辦清單中。
測試文字 4：打電話給客戶討論專案進度

請嘗試選取上面任一段文字，應該會看到：

  • 🔵 藍色通知：「正在添加到待辦清單...」
  • ✅ 綠色通知：「已添加到待辦清單: [文字內容]」
  • ❌ 紅色通知：如果出現錯誤

🔧 診斷工具

💡 故障排除

如果沒有看到通知：

  • 按 F12 打開開發者工具，查看 Console 標籤中的錯誤訊息
  • 確認擴展已正確載入：進入 chrome://extensions/ 檢查
  • 確認權限：擴展應該有 "activeTab", "scripting", "storage", "tabs" 權限
  • 重新載入擴展：在擴展管理頁面點擊重新載入按鈕
  • 重新載入此頁面並再次嘗試

應該看到的控制台日誌：

TodoList Extension: Content script loaded on: [URL]
TodoList Extension: Chrome runtime available
TodoList Extension: Content script initialization complete
TodoList Extension: Selected text: [選取的文字]
TodoList Extension: Message sent successfully
